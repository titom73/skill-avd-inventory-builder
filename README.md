 # AVD Skill
 
 This repository contains “skill” resources and examples for the Arista AVD ecosystem. It provides:
 
 - input and script examples in [docs/examples](docs/examples)
 - schema notes and references in [docs/skills/PYAVD_SCHEMAS.md](docs/skills/PYAVD_SCHEMAS.md)
 - installation instructions in [INSTALL.md](INSTALL.md)
 
 ## Contents
 
 - [docs/examples/eos_designs_minimal_fabric.yml](docs/examples/eos_designs_minimal_fabric.yml): minimal fabric example for eos_designs.
 - [docs/examples/pyavd_hello_world.py](docs/examples/pyavd_hello_world.py): sample Python script for PyAVD.
 - [docs/skills/PYAVD_SCHEMAS.md](docs/skills/PYAVD_SCHEMAS.md): guidance on PyAVD schemas.
 
 ## Installation
 
 Follow the steps in [INSTALL.md](INSTALL.md).
 
 ## Quick start
 
 1. Browse the examples in [docs/examples](docs/examples).
 2. Adapt the example files to your environment.
 3. Refer to the schema notes in [docs/skills/PYAVD_SCHEMAS.md](docs/skills/PYAVD_SCHEMAS.md).
